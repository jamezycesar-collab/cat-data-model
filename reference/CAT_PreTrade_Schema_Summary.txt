1. Scope
--------------------------------------------------------------------------------

The model covers the enterprise trade lifecycle for a U.S. broker-dealer
subject to SEC Rule 613 and FINRA Rule 6800 series, expanded from the
original 29-entity pre-trade compliance schema to ~93 entities across
twelve subject areas.

 Party............ 14 entities
 Instrument....... 14 entities
 Venue............ 4 entities
 Agreement........ 9 entities
 Trade Lifecycle.. 20 entities
 Regulatory....... 6 entities
 Reference........ 16 entities
 ---------------------------
 Total............ 83 transactional + reference + shared ≈ 93

(Counts represent the deduplicated entity set across 01_/01a-c and
02_/02a-c split files; see audit/validate_entity_matrix.csv for the
authoritative per-entity ledger.)

5. Reference tables - 16 tables enforcing 72 enumerations
--------------------------------------------------------------------------------

CAT-taxonomy tables: ref_cat_event_type, ref_submission_file_type
ISO-standard tables: ref_country, ref_currency, ref_mic, ref_cfi_category,
 ref_entity_legal_form
Industry tables: ref_instrument_type, ref_asset_class, ref_market_segment,
 ref_isda_daycount, ref_fix_enum, ref_credit_rating,
 ref_tax_jurisdiction
Firm/regulatory tables: ref_firm_desk, ref_risk_framework

Enumerations are enforced via FOREIGN KEY to the ref table, with CHECK
constraints used only where a small closed set is not a reference table
(e.g., risk_framework).

7. CAT event coverage
--------------------------------------------------------------------------------

All 50 CAT events from the FINRA CAT Industry Member Technical
Specifications v4.1.0r9 taxonomy are mapped to target entities:

 MEIR / MOIR - Order Request (electronic / manual receipt)
 MEQR - Quote Request / RFQ
 MEQS - Quote response
 MENO / MONO - New Order / Manual New Order
 MEMR / MOMR - Route / Manual Route
 MEOT / MEOTQ/ MEOF - Execution (trade / quote-trade / fulfillment)
 MECR / MOCR - Cancel / Replace
 MEAM / MOAM - Allocation
 MECF - Confirmation... and 38 additional events

See audit/validate_entity_matrix.csv for the per-event mapping.
